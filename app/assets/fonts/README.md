# 請求書PDF用 日本語フォント（任意）

このフォルダに日本語 TTF/OTF を置くと、請求書PDFの文字化けを防げます。

**対応ファイル名（いずれか1つで可）**

- `NotoSansCJKjp-Regular.ttf` または `NotoSansCJKjp-Regular.otf`  
  - [Noto Sans CJK JP](https://github.com/googlefonts/noto-cjk) から取得
- `IPAexGothic.ttf`  
  - [IPAフォント](https://moji.or.jp/ipafont/) の「IPAexゴシック」から取得

置かなくても、macOS ではシステムの AppleSDGothicNeo（TTC）を `font: 0` で使用します。
